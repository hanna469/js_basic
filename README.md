# 자바스크립트 260306 ~
* 자바스크립트는 동적 언어로 변수 정의 시 자료형 정의를 할 필요가 없고 대입되는 데이터 종류에 따라 변수의 데이터 종류가 결정됨.
* = :: 대입연산자
* 변수 하나 당 데이터 하나만 대입할 수 있음, 기본값 (새로운 값 대입할 경우 이전값 지워짐)
## 변수
* 프로그래밍의 첫번째 시작작업은 필요한 데이터들을 반드시 변수(저장소)로 저장해야 한다.
## 변수 작성법
* `변수선언키워드 생성변수명;` -> 빈 저장소 생성
    * `var box;`
* `변수선언키워드 생성변수명 대입연산자 대입값;`
    * `var box = 1;`
* `생성변수명 대입연산자 대입값;`
    * `box = 2;`
* 변수선언키워드 : var, let, const
* 생성변수명 - 영문, 숫자조합(숫자는 영문뒤로)
## 변수명
* `var` (변수)
    * 전역 스코프 선언 방식
    * 중복 선언 가능
* `let` (변수)
    * `var` 선언 키워드와 유사하나 중복 선언 불가
    * 블록 스코프 선언 방식
* `const` (상수)
    * 변경되지 않는 상수로 최초 선언 후 값 변경 불가
    * 고정된 변수 데이터에만 const 선언 가능
    * 블록 스코프 선언 방식
### 변수 작성 시 주의사항
* 변수의 이름은 문자(a~z, A~Z), 숫자(0~9), 달러($), 밑줄(_)이 포함됨
* 변수의 이름은 숫자로 시작할 수 없음
* 이름은 대시(-)나 마침표(.) 기호를 사용할 수 없음
* 키워드나 예약어는 사용할 수 없음
    * 키워드, 예약어 종류 -> `delete`, `this`, `false`, `true`, `if`, `default`, `var`, `new`, `class` 등
    * 키워드, 예약어 = js에서 명령어로 정해진 단어
* 모든 변수는 대/소문자를 구별. **`score`와 `Score`는 다른 변수임**
* **변수가 저장할 정보의 종류를 표현하는 의미있는 단어를 사용할 것**
### 변수 표기법 종류
* 캐멀 표기법 | 앞글자 소문자/뒷글자에서 새로운 단어가 나올 때 대문자로 구분
    * `newName`
    * `webSiteCode`
* 파스칼 표기법 | 모든 단어 첫글자를 대문자로 작성
    * `NewName`
    * `WebSiteCode`
* 밑줄 표기법 | 단어 사이에 언더바(_) 작성
    * `new_name`
    * `web_site_code`
## 객체, 속성, 메서드 작성법
* `객체;` -> 객체 읽음
* `객체.속성;` -> 객체의 속성을 읽음
* `객체.메서드();` -> 객체를 메서비방법으로 실행함
* `객체.속성.메서드();`
* `객체.속성.속성.메서드();`
* `객체.메서드().속성;`
* `객체.메서드().메서드();`

### 템플릿문자열
* 뱃틱기호(`)로 전체 문자열을 묶어주고 변수 부분은 ${} 안에 입력하여 작성
* 변수끼리 +,*로 더하기/곱하기 사용 가능
* `${product_item} ${product_quantity}개의 가격은 ${product_price*product_quantity}원 입니다.`
## 데이터타입 구분 함수 `typeof()`
* css의 배경색상으로 영역구분하는 것처럼 데이터타입 구분목적 사용
1. 특정 데이터타입을 담아둔 변수를 생성
    * `let num = 1;`
    * `let str = '1';`
2. 해당 변수의 데이터타입을 구분하기 위해 typeof 함수확인
    * `console.log(typeof(num));` //Number(숫자)
    * `console.log(typeof(str));` //String(문자)
